# Basketball National Team WordPress Website

## Overview

This project is a WordPress-based website dedicated to a national basketball team. It provides dynamic content management and allows administrators to easily update player information, match schedules, statistics, news, and other website content through the WordPress dashboard.

## Features

* Responsive design
* Dynamic content management using WordPress
* Player profiles and team information
* Match schedules and results
* News and updates section
* Easy content updates through the admin panel

## Technologies Used

* WordPress
* PHP
* HTML
* CSS

## Description

This project represents a dynamic version of a basketball team website built using WordPress. Compared to a static website, WordPress enables easier content management through the administration panel without requiring direct code modifications. The website provides information about players, matches, statistics, achievements, and fan engagement features.

## Setup Instructions

* Install WordPress on a local server (XAMPP/WAMP)
* Place the project files in the server directory (htdocs)
* Import the database if required
* Run the project through the browser (localhost)


## Note

This project is the WordPress implementation of the same Basketball National Team website that was also developed using Bootstrap.

A demonstration video of the website can be found in the **View** folder.
